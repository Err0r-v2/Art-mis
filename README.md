#  [WIP] Artemis 🏹


<a href="https://www.zone-telechargement.onl/">Zone-Téléchargement</a> scraper and downloader

##  


### ☕ Usage  
- #### 💻 Downloading
     ```
    >> git clone https://github.com/Err0r-v2/Artemis.git
    >> cd Artemis/
    >> pip install -r requirements.txt
    ```
- #### 🖥️ Starting
      >> python3 main.py

##  

### 🏆 Features List
- Very Fast Checking
- Proxy support: socks 4/5
- Simple Usage
- Direct download
- Supports caching
